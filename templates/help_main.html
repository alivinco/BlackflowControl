The Device class enumeration

The Device class enumeration is limited set of available devices which has to be used as part of URI as well as part of message (origin.device.@type) .

Binary devices

the devices which are binary by nature therefore they either accepts binary command or emits binary event : 
bin_switch
bin_motion
bin_opening
bin_fire
bin_flood
bin_gas
bin_relay
bin_smoke

Mode centric devices :

mod_scenario
mod_siren

Level change devices :

lvl_dimmer
lvl_volume
lvl_thermostat
lvl_battery

Sensors

a sensors is a device which reports instant simple measurement ( instant measurement sample ) , it doesn't do any complex data aggregation or post processing . The reported value has to be atomic and do not have mandatory links to other measurements , in other words it can be used as stand alone measurement :

sen_humidity
sen_gas
sen_genpurpose
sen_lumin
sen_power

Meters

a meter is complex device which performs data aggregation and post processing , normally it consists of sensor + aggregation memory + post processing calculations : 
met_power
met_flow
met_flow
met_heat
met_radiation
met_radon
