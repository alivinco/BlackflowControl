Filtering help

Filtering field supports regular expression syntax .Here are some examples:

/dev/zw/6/ - returns all sensors for device numeber 6

(sen_temp|sen_humidity) - returns all devices which are either sen_temp or sen_humidity

zw/\d*/(sen_temp|sen_humidity) - returns any zwave devices which has sen_temp or sen_humidity sensors

/dev/zw/(35|40|3) - returns all entries for devices nr 35,40,3

\s*/zw/\d\d/bin_open - returns all zwave devices which has bin_open sensor. \s - any character , \d - any number
